# Employee Management System (Spring Boot)

## Features
- Create Employee
- Get All Employees
- Get Employee by ID
- Update Employee
- Delete Employee

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints
POST /api/employees  
GET /api/employees  
GET /api/employees/{id}  
PUT /api/employees/{id}  
DELETE /api/employees/{id}
